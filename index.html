URL to Markdown

您可以点击 MDTU 编辑器右上角链接采集按钮，输入 URL 回车确认可从 URL 中获取 Markdown。

自动编号、插入目录、插入导图

您可以点击 MDTU 编辑器左上角自动编号，将会对一级标题、二级标题、三级标题进行自动编号。

使用模板

您可以点击 MDTU 编辑器 使用模板 -> 管理模板 按钮，配置自己的模板。

使用占位符 {{ content }} 代表正文内容。

方格背景

您可以点击 MDTU 编辑器 UI -> 方格背景，将会对该主题添加上方格背景。
